# REST API services for DocQ-APP

## Prerequisites:

Update JWT secret key and MongoDB url in `.env` file like below.

```
JWT_SECRET=YOUR_SECRET_KEY
MONGODB_CONNECTION_STRING="mongodb+srv://user_name:password@DB_host/DB_name"


```

# How to start server

#### Developement Mode

```
npm run devStart
```

#### Production Mode

```
npm start
```
